X≈WHX≈WH

3. [Spectral clustering, 30 points] In spectral clustering the clustering problem is transformed to a graph partitioning problem. The input data X={x(1),…,x(n)}X={x(1),…,x(n)} is used to construct a similarity graph of pair-wise similarities between data points. The graph is then partitioned into disjoint sets of connected vertices which correspond to clusters in XX. While many clustering algorithm such as kk-means impose strict assumptions on the cluster shape, spectral clustering makes no such assumptions which makes it applicable in many situations. Spectral clustering is fairly computationally demanding which limits its use on large data sets.

The following algorithm (NCUT) can be used to partition the data into two groups AA and BB:

1) Construct an n×nn×n similarity matrix WW with Wij=e−γ||x(i),…,x(j)||2Wij=e−γ||x(i),…,x(j)||2 for i,j=1,…,ni,j=1,…,n (note that the matrix is symmetric). Note that γγ is a hyperparameter that you need to specify.

2) Compute the degree of node i, di=∑nj=1wij, j=1,…,ni, di=∑j=1nwij, j=1,…,n and form the diagonal matrix DD with Dii=diDii=di and Dij=0Dij=0 for i≠ji≠j.

3) Compute the unnormalized graph Laplacian L=D−WL=D−W.

4) Solve the generalized eigenvalue problem Lu=λDuLu=λDu. The second smallest eigenvector (in terms of corresponding eigenvalue) u2u2 gives a partition of the data into groups AA and BB as follows. If element ii of u2u2 is negative then x(i)x(i) belongs to group AA and BB otherwise.

For k>2k>2 groups add the following extra step:

5) Form the n×kn×k matrix ZZ with the kk eigenvectors u2,…,uk+1u2,…,uk+1. Apply kk-means to this matrix and return the resulting clustering (the rows of ZZ corresponds to derived features).

3) It is assumed that the eigenvalues are in ascending order and when werefer to the jj-th smallest eigenvector we refer to the eigenvector that corresponds to the jj-th smallest eigenvalue. def ncut(X,k,gamma):
    n,p = X.shape
    gamma=1
    W = np.exp(-gamma*distance.cdist(X, X, 'euclidean')**2)
    d = np.zeros(n)
    for i in range(n):
        d[i] = np.sum(W[i,:])
    D = np.eye(n)*d
    L=D-W
    lam, u = eigh(L, D, eigvals_only=False,eigvals=(1,k+1))
    return(lam,u)
